        еңЁJedisClusterеҲқе§ӢеҢ–зҡ„ж—¶еҖҷпјҢе°ұдјҡйҡҸжңәйҖүжӢ©дёҖдёӘnodeпјҢеҲқе§ӢеҢ–hashslot -> nodeжҳ е°„иЎЁпјҢеҗҢж—¶дёәжҜҸдёӘиҠӮзӮ№еҲӣе»әдёҖдёӘJedisPoolиҝһжҺҘжұ 

        жҜҸж¬ЎеҹәдәҺJedisClusterжү§иЎҢж“ҚдҪңпјҢйҰ–е…ҲJedisClusterйғҪдјҡеңЁжң¬ең°и®Ўз®—keyзҡ„hashslotпјҢз„¶еҗҺеңЁжң¬ең°жҳ е°„иЎЁжүҫеҲ°еҜ№еә”зҡ„иҠӮзӮ№

        еҰӮжһңйӮЈдёӘnodeжӯЈеҘҪиҝҳжҳҜжҢҒжңүйӮЈдёӘhashslotпјҢйӮЈд№Ҳе°ұok; еҰӮжһңиҜҙиҝӣиЎҢдәҶreshardиҝҷж ·зҡ„ж“ҚдҪңпјҢеҸҜиғҪhashslotе·Із»ҸдёҚеңЁйӮЈдёӘnodeдёҠдәҶпјҢе°ұдјҡиҝ”еӣһmoved

        еҰӮжһңJedisCluter APIеҸ‘зҺ°еҜ№еә”зҡ„иҠӮзӮ№иҝ”еӣһmovedпјҢйӮЈд№ҲеҲ©з”ЁиҜҘиҠӮзӮ№зҡ„е…ғж•°жҚ®пјҢжӣҙж–°жң¬ең°зҡ„hashslot -> nodeжҳ е°„иЎЁзј“еӯҳ

        йҮҚеӨҚдёҠйқўеҮ дёӘжӯҘйӘӨпјҢзӣҙеҲ°жүҫеҲ°еҜ№еә”зҡ„иҠӮзӮ№пјҢеҰӮжһңйҮҚиҜ•и¶…иҝҮ5ж¬ЎпјҢйӮЈд№Ҳе°ұжҠҘй”ҷпјҢJedisClusterMaxRedirectionException

        jedisиҖҒзүҲжң¬пјҢеҸҜиғҪдјҡеҮәзҺ°еңЁйӣҶзҫӨжҹҗдёӘиҠӮзӮ№ж•…йҡңиҝҳжІЎе®ҢжҲҗиҮӘеҠЁеҲҮжҚўжҒўеӨҚж—¶пјҢйў‘з№Ғжӣҙж–°hash slotпјҢйў‘з№ҒpingиҠӮзӮ№жЈҖжҹҘжҙ»и·ғпјҢеҜјиҮҙеӨ§йҮҸзҪ‘з»ңIOејҖй”Җ

        jedisжңҖж–°зүҲжң¬пјҢеҜ№дәҺиҝҷдәӣиҝҮеәҰзҡ„hash slotжӣҙж–°е’ҢpingпјҢйғҪиҝӣиЎҢдәҶдјҳеҢ–пјҢйҒҝе…ҚдәҶзұ»дјјй—®йўҳ

        пјҲ3пјүhashslotиҝҒз§»е’ҢaskйҮҚе®ҡеҗ‘

        еҰӮжһңhash slotжӯЈеңЁиҝҒз§»пјҢйӮЈд№Ҳдјҡиҝ”еӣһaskйҮҚе®ҡеҗ‘з»ҷjedis

        jedisжҺҘж”¶еҲ°askйҮҚе®ҡеҗ‘д№ӢеҗҺпјҢдјҡйҮҚж–°е®ҡдҪҚеҲ°зӣ®ж ҮиҠӮзӮ№еҺ»жү§иЎҢпјҢдҪҶжҳҜеӣ дёәaskеҸ‘з”ҹеңЁhash slotиҝҒз§»иҝҮзЁӢдёӯпјҢжүҖд»ҘJedisCluster APIж”¶еҲ°askжҳҜдёҚдјҡжӣҙж–°hashslotжң¬ең°зј“еӯҳ

        е·Із»ҸеҸҜд»ҘзЎ®е®ҡиҜҙпјҢhashslotе·Із»ҸиҝҒз§»е®ҢдәҶпјҢmovedжҳҜдјҡжӣҙж–°жң¬ең°hashslot->nodeжҳ е°„иЎЁзј“еӯҳзҡ„

        -------------------------------------------------------------------------------------------------------

дёүгҖҒй«ҳеҸҜз”ЁжҖ§дёҺдё»еӨҮеҲҮжҚўеҺҹзҗҶ

        redis clusterзҡ„й«ҳеҸҜз”Ёзҡ„еҺҹзҗҶпјҢеҮ д№Һи·ҹе“Ёе…өжҳҜзұ»дјјзҡ„

        1гҖҒеҲӨж–ӯиҠӮзӮ№е®•жңә

        еҰӮжһңдёҖдёӘиҠӮзӮ№и®ӨдёәеҸҰеӨ–дёҖдёӘиҠӮзӮ№е®•жңәпјҢйӮЈд№Ҳе°ұжҳҜpfailпјҢдё»и§Ӯе®•жңә

        еҰӮжһңеӨҡдёӘиҠӮзӮ№йғҪи®ӨдёәеҸҰеӨ–дёҖдёӘиҠӮзӮ№е®•жңәдәҶпјҢйӮЈд№Ҳе°ұжҳҜfailпјҢе®ўи§Ӯе®•жңәпјҢи·ҹе“Ёе…өзҡ„еҺҹзҗҶеҮ д№ҺдёҖж ·пјҢsdownпјҢodown

        еңЁcluster-node-timeoutеҶ…пјҢжҹҗдёӘиҠӮзӮ№дёҖзӣҙжІЎжңүиҝ”еӣһpongпјҢйӮЈд№Ҳе°ұиў«и®Өдёәpfail

        еҰӮжһңдёҖдёӘиҠӮзӮ№и®ӨдёәжҹҗдёӘиҠӮзӮ№pfailдәҶпјҢйӮЈд№ҲдјҡеңЁgossip pingж¶ҲжҒҜдёӯпјҢpingз»ҷе…¶д»–иҠӮзӮ№пјҢеҰӮжһңи¶…иҝҮеҚҠж•°зҡ„иҠӮзӮ№йғҪи®ӨдёәpfailдәҶпјҢйӮЈд№Ҳе°ұдјҡеҸҳжҲҗfail

        2гҖҒд»ҺиҠӮзӮ№иҝҮж»Ө

        еҜ№е®•жңәзҡ„master nodeпјҢд»Һе…¶жүҖжңүзҡ„slave nodeдёӯпјҢйҖүжӢ©дёҖдёӘеҲҮжҚўжҲҗmaster node

        жЈҖжҹҘжҜҸдёӘslave nodeдёҺmaster nodeж–ӯејҖиҝһжҺҘзҡ„ж—¶й—ҙпјҢеҰӮжһңи¶…иҝҮдәҶcluster-node-timeout * cluster-slave-validity-factorпјҢйӮЈд№Ҳе°ұжІЎжңүиө„ж јеҲҮжҚўжҲҗmaster

        иҝҷдёӘд№ҹжҳҜи·ҹе“Ёе…өжҳҜдёҖж ·зҡ„пјҢд»ҺиҠӮзӮ№и¶…ж—¶иҝҮж»Өзҡ„жӯҘйӘӨ

        3гҖҒд»ҺиҠӮзӮ№йҖүдёҫ

        е“Ёе…өпјҡеҜ№жүҖжңүд»ҺиҠӮзӮ№иҝӣиЎҢжҺ’еәҸпјҢslave priorityпјҢoffsetпјҢrun id

        жҜҸдёӘд»ҺиҠӮзӮ№пјҢйғҪж №жҚ®иҮӘе·ұеҜ№masterеӨҚеҲ¶ж•°жҚ®зҡ„offsetпјҢжқҘи®ҫзҪ®дёҖдёӘйҖүдёҫж—¶й—ҙпјҢoffsetи¶ҠеӨ§пјҲеӨҚеҲ¶ж•°жҚ®и¶ҠеӨҡпјүзҡ„д»ҺиҠӮзӮ№пјҢйҖүдёҫж—¶й—ҙи¶Ҡйқ еүҚпјҢдјҳе…ҲиҝӣиЎҢйҖүдёҫ

        жүҖжңүзҡ„master nodeејҖе§ӢslaveйҖүдёҫжҠ•зҘЁпјҢз»ҷиҰҒиҝӣиЎҢйҖүдёҫзҡ„slaveиҝӣиЎҢжҠ•зҘЁпјҢеҰӮжһңеӨ§йғЁеҲҶmaster nodeпјҲN/2 + 1пјүйғҪжҠ•зҘЁз»ҷдәҶжҹҗдёӘд»ҺиҠӮзӮ№пјҢйӮЈд№ҲйҖүдёҫйҖҡиҝҮпјҢйӮЈдёӘд»ҺиҠӮзӮ№еҸҜд»ҘеҲҮжҚўжҲҗmaster

        д»ҺиҠӮзӮ№жү§иЎҢдё»еӨҮеҲҮжҚўпјҢд»ҺиҠӮзӮ№еҲҮжҚўдёәдё»иҠӮзӮ№

        4гҖҒдёҺе“Ёе…өжҜ”иҫғ

        ж•ҙдёӘжөҒзЁӢи·ҹе“Ёе…өзӣёжҜ”пјҢйқһеёёзұ»дјјпјҢжүҖд»ҘиҜҙпјҢredis clusterеҠҹиғҪејәеӨ§пјҢзӣҙжҺҘйӣҶжҲҗдәҶreplicationе’Ңsentinalзҡ„еҠҹиғҪ


жІЎжңүеҠһжі•еҺ»з»ҷеӨ§е®¶ж·ұе…Ҙи®Іи§Јredisеә•еұӮзҡ„и®ҫи®Ўзҡ„з»ҶиҠӮпјҢж ёеҝғеҺҹзҗҶе’Ңи®ҫи®Ўзҡ„з»ҶиҠӮпјҢйӮЈдёӘйҷӨйқһеҚ•зӢ¬ејҖдёҖй—ЁиҜҫпјҢredisеә•еұӮеҺҹзҗҶж·ұеәҰеү–жһҗпјҢredisжәҗз Ғ

еҜ№дәҺе’ұд»¬иҝҷдёӘжһ¶жһ„иҜҫжқҘиҜҙпјҢдё»иҰҒе…іжіЁзҡ„жҳҜжһ¶жһ„пјҢдёҚжҳҜеә•еұӮзҡ„з»ҶиҠӮпјҢеҜ№дәҺжһ¶жһ„жқҘиҜҙпјҢж ёеҝғзҡ„еҺҹзҗҶзҡ„еҹәжң¬жҖқи·ҜпјҢжҳҜиҰҒжўізҗҶжё…жҷ°зҡ„
