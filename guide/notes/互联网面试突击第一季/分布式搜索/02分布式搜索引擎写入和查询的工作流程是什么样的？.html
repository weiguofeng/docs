# еҲҶеёғејҸжҗңзҙўеј•ж“ҺеҶҷе…Ҙе’ҢжҹҘиҜўзҡ„е·ҘдҪңжөҒзЁӢжҳҜд»Җд№Ҳж ·зҡ„пјҹ

# 1.йқўиҜ•йўҳ

вҖӢ 2.esеҶҷе…Ҙж•°жҚ®зҡ„е·ҘдҪңеҺҹзҗҶжҳҜд»Җд№Ҳе•ҠпјҹesжҹҘиҜўж•°жҚ®зҡ„е·ҘдҪңеҺҹзҗҶжҳҜд»Җд№Ҳе•Ҡпјҹеә•еұӮзҡ„lunceд»Ӣз»ҚдёҖдёӢе‘—пјҹеҖ’жҺ’зҙўеј•дәҶи§Јеҗ—пјҹ

# 2.йқўиҜ•е®ҳеҝғйҮҢеҲҶжһҗ

	й—®иҝҷдёӘе…¶е®һйқўиҜ•е®ҳе°ұжҳҜжғізңӢзңӢдҪ дәҶи§ЈдёҚдәҶи§Јesзҡ„дёҖдәӣеҹәжң¬еҺҹзҗҶпјҢеӣ дёәз”Ёesж— йқһе°ұжҳҜеҶҷе…Ҙж•°жҚ®пјҢжҗңзҙўж•°жҚ®пјҢдҪ иҰҒжҳҜдҪ еҸ‘иө·дёҖдёӘеҶҷе…Ҙе’ҢжҗңзҙўиҜ·жұӮзҡ„ж—¶еҖҷпјҢesеңЁе№Ід»Җд№ҲпјҢйӮЈдҪ зңҹзҡ„е°ұжҳҜoutдәҶгҖӮ
	
	
	еҜ№esеҹәжң¬е°ұжҳҜдёӘй»‘зӣ’пјҢдҪ иҝҳиғҪе№Іе•ҘпјҹдҪ е”ҜдёҖиғҪе№Ізҡ„е°ұжҳҜз”Ёesзҡ„apiиҜ»еҶҷж•°жҚ®дәҶпјҢиҰҒжҳҜеҮәзӮ№д»Җд№Ҳй—®йўҳпјҢдҪ е•ҘйғҪдёҚзҹҘйҒ“пјҢйӮЈиҝҳиғҪжҢҮжңӣдҪ е№Ід»Җд№Ҳе‘ўпјҹжҳҜдёҚжҳҜ

# 3.йқўиҜ•йўҳеү–жһҗ

	1.ESеҶҷж•°жҚ®зҡ„иҝҮзЁӢ
		1.1пјҡе®ўжҲ·з«ҜйҖүжӢ©дёҖдёӘnodeеҸ‘йҖҒиҜ·жұӮиҝҮеҺ»пјҢиҝҷдёӘnodeе°ұжҳҜcoordinating nodeеҚҸи°ғиҠӮзӮ№
		1.2пјҡcoordinating nodeпјҢеҜ№doucumentиҝӣиЎҢи·Ҝз”ұпјҢе°ҶиҜ·жұӮиҪ¬еҸ‘з»ҷеҜ№еә”зҡ„nodeпјҲprimary shardпјү
		1.3пјҡеҜ№еә”зҡ„nodeдёҠзҡ„ primary shard еӨ„зҗҶиҜ·жұӮпјҢз„¶еҗҺе°Ҷж•°жҚ®еҗҢжӯҘеҲ°replic shardдёҠйқўеҺ»гҖӮ
		1.4пјҡcoordinating nodeпјҢеҰӮжһңеҸ‘зҺ°primary nodeе’Ңreplic nodeйғҪжҗһе®ҡд№ӢеҗҺпјҢдјҡиҝ”еӣһе“Қеә”з»ҷе®ўжҲ·з«ҜгҖӮ
		
	
	2.esиҜ»ж•°жҚ®иҝҮзЁӢпјҡ
		жҹҘиҜўпјҢGETжҹҗдёҖжқЎж•°жҚ®пјҢеҶҷе…ҘдәҶжҹҗдёӘdocumentпјҢиҝҷдёӘdocumentдјҡиҮӘеҠЁз»ҷдҪ еҲҶй…ҚдёҖдёӘе…ЁеұҖе”ҜдёҖзҡ„idпјҢdoc idпјҢеҗҢж—¶д№ҹжҳҜж №жҚ®doc idиҝӣиЎҢhashи·Ҝз”ұеҲ°еҜ№еә”зҡ„primary shardдёҠйқўеҺ»гҖӮд№ҹеҸҜд»ҘжүӢеҠЁжҢҮе®ҡdoc idпјҢжҜ”еҰӮз”Ёи®ўеҚ•idпјҢз”ЁжҲ·idгҖӮ

        дҪ еҸҜд»ҘйҖҡиҝҮdoc idжқҘжҹҘиҜўпјҢдјҡж №жҚ®doc idиҝӣиЎҢhashпјҢеҲӨж–ӯеҮәжқҘеҪ“ж—¶жҠҠdoc idеҲҶй…ҚеҲ°дәҶе“ӘдёӘshardдёҠйқўеҺ»пјҢд»ҺйӮЈдёӘshardеҺ»жҹҘиҜў

        1пјүе®ўжҲ·з«ҜеҸ‘йҖҒиҜ·жұӮеҲ°д»»ж„ҸдёҖдёӘnodeпјҢжҲҗдёәcoordinate node
        2пјүcoordinate nodeеҜ№documentиҝӣиЎҢи·Ҝз”ұпјҢе°ҶиҜ·жұӮиҪ¬еҸ‘еҲ°еҜ№еә”зҡ„nodeпјҢжӯӨж—¶дјҡдҪҝз”Ёround-robinйҡҸжңәиҪ®иҜўз®—жі•пјҢеңЁprimary shardд»ҘеҸҠе…¶жүҖжңүreplicaдёӯйҡҸжңәйҖүжӢ©дёҖдёӘпјҢи®©иҜ»иҜ·жұӮиҙҹиҪҪеқҮиЎЎ
        3пјүжҺҘж”¶иҜ·жұӮзҡ„nodeиҝ”еӣһdocumentз»ҷcoordinate node
        4пјүcoordinate nodeиҝ”еӣһdocumentз»ҷе®ўжҲ·з«Ҝ

	3гҖӮesжҗңзҙўж•°жҚ®иҝҮзЁӢ

        esжңҖејәеӨ§зҡ„жҳҜеҒҡе…Ёж–ҮжЈҖзҙўпјҢе°ұжҳҜжҜ”еҰӮдҪ жңүдёүжқЎж•°жҚ®

        javaзңҹеҘҪзҺ©е„ҝе•Ҡ
        javaеҘҪйҡҫеӯҰе•Ҡ
        j2eeзү№еҲ«зүӣ

        дҪ ж №жҚ®javaе…ій”®иҜҚжқҘжҗңзҙўпјҢе°ҶеҢ…еҗ«javaзҡ„documentз»ҷжҗңзҙўеҮәжқҘ

        esе°ұдјҡз»ҷдҪ иҝ”еӣһпјҡjavaзңҹеҘҪзҺ©е„ҝе•ҠпјҢjavaеҘҪйҡҫеӯҰе•Ҡ

        1пјүе®ўжҲ·з«ҜеҸ‘йҖҒиҜ·жұӮеҲ°дёҖдёӘcoordinate node
        2пјүеҚҸи°ғиҠӮзӮ№е°ҶжҗңзҙўиҜ·жұӮиҪ¬еҸ‘еҲ°жүҖжңүзҡ„shardеҜ№еә”зҡ„primary shardжҲ–replica shardд№ҹеҸҜд»Ҙ
        3пјүquery phaseпјҡжҜҸдёӘshardе°ҶиҮӘе·ұзҡ„жҗңзҙўз»“жһңпјҲе…¶е®һе°ұжҳҜдёҖдәӣdoc idпјүпјҢиҝ”еӣһз»ҷеҚҸи°ғиҠӮзӮ№пјҢз”ұеҚҸи°ғиҠӮзӮ№иҝӣиЎҢж•°жҚ®зҡ„еҗҲе№¶гҖҒжҺ’еәҸгҖҒеҲҶйЎөзӯүж“ҚдҪңпјҢдә§еҮәжңҖз»Ҳз»“жһң
        4пјүfetch phaseпјҡжҺҘзқҖз”ұеҚҸи°ғиҠӮзӮ№пјҢж №жҚ®doc idеҺ»еҗ„дёӘиҠӮзӮ№дёҠжӢүеҸ–е®һйҷ…зҡ„documentж•°жҚ®пјҢжңҖз»Ҳиҝ”еӣһз»ҷе®ўжҲ·з«Ҝ
	
	4.жҗңзҙўзҡ„еә•еұӮеҺҹзҗҶпјҢеҖ’жҺ’зҙўеј•пјҢз”»еӣҫиҜҙжҳҺдј з»ҹж•°жҚ®еә“е’ҢеҖ’жҺ’зҙўеј•зҡ„еҢәеҲ«
	
	
	
	
	5.еҶҷж•°жҚ®зҡ„еә•еұӮеҺҹзҗҶпјҲе…¶е®һе°ұжҳҜshardйҮҢзҡ„еҶҷе…ҘиҝҮзЁӢпјү
		5.1пјҡе…ҲеҶҷе…ҘbufferпјҢеңЁbufferйҮҢзҡ„ж—¶еҖҷж•°жҚ®жҳҜжҗңзҙўдёҚеҲ°зҡ„пјҢ
		5.2пјҡеҰӮжһңbufferеҝ«ж»ЎдәҶпјҢжҲ–иҖ…еҲ°дёҖе®ҡж—¶й—ҙеҗҺпјҢbufferе°Ҷж•°жҚ®еҶҷе…ҘеҲ° os cacheпјҲж“ҚдҪңзі»з»ҹзј“еӯҳпјүпјҢз„¶еҗҺд»Һos cache еҶҷе…Ҙ segment fileзЈҒзӣҳж–Үд»¶дёӯпјҢиҝҷдёӘиҝҮзЁӢе°ұжҳҜrefreshпјҢејәеҲ¶еҲ·ж–°гҖӮ
		
			жҜҸйҡ”1sпјҢesе°Ҷbufferдёӯзҡ„ж•°жҚ®еҶҷе…ҘдёҖдёӘж–°зҡ„segment fileдёӯпјҢжҜҸз§’й’ҹдјҡдә§з”ҹдёҖдёӘж–°зҡ„зЈҒзӣҳж–Үд»¶пјҢиҝҷдёӘsegment fileе°ұеӯҳеӮЁжңҖиҝ‘1sеҶ…зҡ„buffer дёӯеҶҷе…Ҙзҡ„ж•°жҚ®гҖӮ
			дҪҶжҳҜеҰӮжһңbufferдёӯжІЎжңүж•°жҚ®пјҢйӮЈеҪ“з„¶дёҚдјҡжү§иЎҢrefreshж“ҚдҪңе’ҜпјҢеӣ дёәжҜҸж¬ЎrefreshдёҖж¬ЎйғҪз”ҹжҲҗдёҖдёӘsegment fileпјҢеҰӮжһңжҳҜз©әзҡ„жҳҜжІЎжңүж„Ҹд№үзҡ„гҖӮеҰӮжһңbufferйҮҢжңүж•°жҚ®пјҢйӮЈд№Ҳй»ҳи®ӨжҳҜ1sжү§иЎҢдёҖж¬ЎrefreshпјҢе°Ҷж•°жҚ®еҲ·е…Ҙsegment fileдёӯ
			еҸӘиҰҒbufferдёӯзҡ„ж•°жҚ®иў«refreshж“ҚдҪңпјҢеҲ·е…Ҙos cache дёӯпјҢе°ұд»ЈиЎЁиҝҷдёӘж•°жҚ®еҸҜд»Ҙиў«жҗңзҙўеҲ°дәҶгҖӮ
			дёәд»Җд№ҲеҸ«esжҳҜеҮҶе®һж—¶зҡ„пјҹNRTпјҢnear real-timeпјҢеҮҶе®һж—¶пјҢй»ҳи®ӨжҳҜжҜҸйҡ”1s refreshдёҖж¬ЎпјҢжүҖд»ҘesжҳҜеҮҶе®һж—¶зҡ„пјҢеӣ дёәеҶҷе…Ҙж•°жҚ®1sд№ӢеҗҺе°ұеҸҜд»Ҙиў«жҹҘиҜўеҲ°дәҶгҖӮ
			
			еҸҜд»ҘйҖҡиҝҮesзҡ„restful apiжҲ–иҖ…java apiпјҢжүӢеҠЁжү§иЎҢдёҖж¬Ўrefreshж“ҚдҪңпјҢе°ұжҳҜжүӢеҠЁе°Ҷж•°жҚ®еҲ·е…Ҙ os cacheз„¶еҗҺз”ҹжҲҗsegment fileзЈҒзӣҳж–Үд»¶пјҢи®©ж•°жҚ®з«Ӣй©¬е°ұиў«жҗңзҙўеҲ°пјҲжҜ”еҰӮеҒҡе®Ңе…Ёе®һж—¶пјү
			
			еҸӘиҰҒж•°жҚ®иў«еҲ·е…ҘеҲ°os cacheдёӯпјҢbufferе°ұиў«жё…з©әдәҶпјҢеӣ дёәдёҚйңҖиҰҒдҝқз•ҷbufferдәҶпјҢж•°жҚ®еңЁtranslogйҮҢйқўе·Із»ҸжҢҒд№…еҢ–еҲ°зЈҒзӣҳйҮҢйқўеҺ»дёҖд»ҪдәҶгҖӮ
			
		5.3пјҡеҸӘиҰҒж•°жҚ®иҝӣе…Ҙos cacheпјҢжӯӨж—¶е°ұеҸҜд»Ҙи®©иҝҷдёӘsegment fileзҡ„ж•°жҚ®еҜ№еӨ–жҸҗдҫӣжҗңзҙўдәҶгҖӮ
		5.4пјҡйҮҚеӨҚ1-3жӯҘйӘӨпјҢж–°зҡ„ж•°жҚ®дёҚж–ӯиҝӣе…Ҙbufferз„¶еҗҺиҝӣе…ҘtranslogпјҢз„¶еҗҺдёҚж–ӯе°Ҷbufferж•°жҚ®еҶҷе…ҘдёҖдёӘеҸҲдёҖдёӘж–°зҡ„segment fileдёӯеҺ»пјҢжҜҸж¬Ўrefreshе®Ңbufferиў«жё…з©әпјҢtranslogдҝқз•ҷи®°еҪ•пјҢйҡҸзқҖиҝҷдёӘиҝҮзЁӢжҺЁиҝӣпјҢtranslogдјҡеҸҳзҡ„и¶ҠжқҘи¶ҠеӨ§пјҢеҪ“translogиҫҫеҲ°дёҖе®ҡй•ҝеәҰж—¶пјҢе°ұдјҡи§ҰеҸ‘commitж“ҚдҪңгҖӮ
			
			bufferдёӯзҡ„ж•°жҚ®пјҢеҖ’жҳҜеҘҪпјҢжҜҸйҡ”1sе°ұиў«еҲ·еҲ°os cacheдёӯеҺ»дәҶпјҢз„¶еҗҺиҝҷдёӘbufferиў«жё…з©әпјҢжүҖд»ҘиҝҷдёӘbufferзҡ„ж•°жҚ®е§Ӣз»ҲжҳҜеҸҜд»ҘдҝқжҢҒдёҚдјҡеЎ«ж»ЎesиҝӣзЁӢзҡ„еҶ…еӯҳзҡ„гҖӮ
		5.5пјҡcommitж“ҚдҪңеҸ‘з”ҹз¬¬дёҖжӯҘпјҢе°ұжҳҜе°ҶbufferдёӯзҺ°жңүж•°жҚ®refreshеҲ°os cacheдёӯпјҢжё…з©әbuffer
		5.6пјҡе°ҶдёҖдёӘ commit pointеҶҷе…ҘзЈҒзӣҳж–Үд»¶пјҢйҮҢйқўж ҮиҜҶзқҖиҝҷдёӘcommit pointеҜ№еә”зҡ„жүҖжңүsegment fileгҖӮ
		5.7пјҡејәиЎҢе°Ҷ os cacheдёӯзҡ„ж•°жҚ®йғҪ fsync еҲ°зЈҒзӣҳж–Үд»¶дёӯгҖӮ
		
			tranlogж—Ҙеҝ—ж–Үд»¶жңүд»Җд№ҲдҪңз”Ёпјҹе°ұжҳҜдҪ еңЁжү§иЎҢcommitд№ӢеүҚпјҢж•°жҚ®иҰҒд№ҲеҒңз•ҷеңЁbufferдёӯпјҢиҰҒд№ҲеҒңз•ҷеңЁos cacheдёӯпјҢж— и®әжҳҜbufferиҝҳжҳҜos cacheйғҪжҳҜеҶ…еӯҳпјҢдёҖж—ҰиҝҷеҸ°жңәеҷЁе®•жңәдәҶпјҢеҶ…еӯҳдёӯзҡ„ж•°жҚ®е°ұдёўеӨұдәҶгҖӮ
			жүҖд»ҘйңҖиҰҒе°Ҷж•°жҚ®еҜ№еә”зҡ„ж“ҚдҪңеҶҷе…ҘдёҖдёӘдё“й—Ёзҡ„ж—Ҙеҝ—ж–Үд»¶пјҢtranslogж–Үд»¶дёӯпјҢдёҖж—ҰжӯӨж—¶жңәеҷЁе®•жңәпјҢеҶҚж¬ЎйҮҚеҗҜзҡ„ж—¶еҖҷпјҢesдјҡиҮӘеҠЁиҜ»еҸ–translogж—Ҙеҝ—ж–Үд»¶дёӯзҡ„ж•°жҚ®пјҢжҒўеӨҚеҲ°bufferе’Ңos cacheдёӯеҺ»гҖӮ
			commitж“ҚдҪңпјҡ1гҖҒеҶҷcommit pointпјӣ2гҖҒе°Ҷos cacheж•°жҚ®fsyncејәеҲ·еҲ°зЈҒзӣҳдёҠеҺ»пјӣ3гҖҒжё…з©әtranslogж—Ҙеҝ—ж–Үд»¶
		
		5.8пјҡе°ҶзҺ°жңүзҡ„translogжё…з©әпјҢз„¶еҗҺеҶҚж¬ЎйҮҚеҗҜеҗҜз”ЁдёҖдёӘtranslogпјҢжӯӨж—¶commitж“ҚдҪңе®ҢжҲҗгҖӮй»ҳи®ӨжҜҸйҡ”30еҲҶй’ҹдјҡиҮӘеҠЁжү§иЎҢдёҖж¬ЎcommitпјҢдҪҶжҳҜеҰӮжһңtranslogиҝҮеӨ§пјҢд№ҹдјҡи§ҰеҸ‘commitгҖӮж•ҙдёӘcommitзҡ„иҝҮзЁӢпјҢеҸ«еҒҡflushж“ҚдҪңгҖӮжҲ‘д»¬еҸҜд»ҘжүӢеҠЁжү§иЎҢflushж“ҚдҪңпјҢе°ұжҳҜе°ҶжүҖжңүos cacheж•°жҚ®еҲ·еҲ°зЈҒзӣҳж–Үд»¶дёӯеҺ»гҖӮ
			
			дёҚеҸ«еҒҡcommitж“ҚдҪңпјҢflushж“ҚдҪңгҖӮesдёӯзҡ„flushж“ҚдҪңпјҢе°ұеҜ№еә”зқҖcommitзҡ„е…ЁиҝҮзЁӢгҖӮжҲ‘д»¬д№ҹеҸҜд»ҘйҖҡиҝҮes apiпјҢжүӢеҠЁжү§иЎҢflushж“ҚдҪңпјҢжүӢеҠЁе°Ҷos cacheдёӯзҡ„ж•°жҚ®fsyncејәеҲ·еҲ°зЈҒзӣҳдёҠеҺ»пјҢи®°еҪ•дёҖдёӘcommit pointпјҢжё…з©әtranslogж—Ҙеҝ—ж–Үд»¶гҖӮ
			
		5.9пјҡtranslogе…¶е®һд№ҹжҳҜе…ҲеҶҷе…Ҙos cacheзҡ„пјҢй»ҳи®ӨжҜҸйҡ”5з§’еҲ·дёҖж¬ЎеҲ°зЈҒзӣҳдёӯеҺ»пјҢжүҖд»Ҙй»ҳи®Өжғ…еҶөдёӢпјҢеҸҜиғҪжңү5з§’зҡ„ж•°жҚ®дјҡд»…д»…еҒңз•ҷеңЁbufferжҲ–иҖ…translogж–Үд»¶зҡ„os cacheдёӯпјҢеҰӮжһңжӯӨж—¶жңәеҷЁжҢӮдәҶпјҢдјҡдёўеӨұ5з§’й’ҹзҡ„ж•°жҚ®гҖӮдҪҶжҳҜиҝҷж ·жҖ§иғҪжҜ”иҫғеҘҪпјҢжңҖеӨҡдёў5з§’зҡ„ж•°жҚ®гҖӮд№ҹеҸҜд»Ҙе°Ҷtranslogи®ҫзҪ®жҲҗжҜҸж¬ЎеҶҷж“ҚдҪңеҝ…йЎ»жҳҜзӣҙжҺҘfsyncеҲ°зЈҒзӣҳпјҢдҪҶжҳҜжҖ§иғҪдјҡе·®еҫҲеӨҡгҖӮ
		
			е®һйҷ…дёҠдҪ еңЁиҝҷйҮҢпјҢеҰӮжһңйқўиҜ•е®ҳжІЎжңүй—®дҪ esдёўж•°жҚ®зҡ„й—®йўҳпјҢдҪ еҸҜд»ҘеңЁиҝҷйҮҢз»ҷйқўиҜ•е®ҳзӮ«дёҖжҠҠпјҢдҪ иҜҙпјҢе…¶е®һesз¬¬дёҖжҳҜеҮҶе®һж—¶зҡ„пјҢж•°жҚ®еҶҷе…Ҙ1з§’еҗҺеҸҜд»ҘжҗңзҙўеҲ°пјӣеҸҜиғҪдјҡдёўеӨұж•°жҚ®зҡ„пјҢдҪ зҡ„ж•°жҚ®жңү5з§’зҡ„ж•°жҚ®пјҢеҒңз•ҷеңЁbufferгҖҒtranslog os cacheгҖҒsegment file os cacheдёӯпјҢжңү5з§’зҡ„ж•°жҚ®дёҚеңЁзЈҒзӣҳдёҠпјҢжӯӨж—¶еҰӮжһңе®•жңәпјҢдјҡеҜјиҮҙ5з§’зҡ„ж•°жҚ®дёўеӨұгҖӮ
			
			еҰӮжһңдҪ еёҢжңӣдёҖе®ҡдёҚиғҪдёўеӨұж•°жҚ®зҡ„иҜқпјҢдҪ еҸҜд»Ҙи®ҫзҪ®дёӘеҸӮж•°пјҢе®ҳж–№ж–ҮжЎЈпјҢзҷҫеәҰдёҖдёӢгҖӮжҜҸж¬ЎеҶҷе…ҘдёҖжқЎж•°жҚ®пјҢйғҪжҳҜеҶҷе…ҘbufferпјҢеҗҢж—¶еҶҷе…ҘзЈҒзӣҳдёҠзҡ„translogпјҢдҪҶжҳҜиҝҷдјҡеҜјиҮҙеҶҷжҖ§иғҪгҖҒеҶҷе…ҘеҗһеҗҗйҮҸдјҡдёӢйҷҚдёҖдёӘж•°йҮҸзә§гҖӮжң¬жқҘдёҖз§’й’ҹеҸҜд»ҘеҶҷ2000жқЎпјҢзҺ°еңЁдҪ дёҖз§’й’ҹеҸӘиғҪеҶҷ200жқЎпјҢйғҪжңүеҸҜиғҪгҖӮ
			
		5.10пјҡеҰӮжһңжҳҜеҲ йҷӨж“ҚдҪңпјҢcommitзҡ„ж—¶еҖҷдјҡз”ҹжҲҗдёҖдёӘ.delж–Үд»¶пјҢйҮҢйқўе°ҶжҹҗдёӘdocж ҮиҜҶдёәdeletedзҠ¶жҖҒпјҢйӮЈд№Ҳжҗңзҙўзҡ„ж—¶еҖҷж №жҚ®.delж–Үд»¶е°ұзҹҘйҒ“иҝҷдёӘdocиў«еҲ йҷӨдәҶгҖӮ
		
		5.11пјҡеҰӮжһңжҳҜжӣҙж–°ж“ҚдҪңпјҢе°ұжҳҜе°ҶеҺҹжқҘзҡ„docж ҮиҜҶдёәdeletedзҠ¶жҖҒпјҢз„¶еҗҺж–°еҶҷе…ҘдёҖжқЎж•°жҚ®
		
		5.12пјҡbufferжҜҸж¬ЎrefreshдёҖж¬ЎпјҢе°ұдјҡдә§з”ҹдёҖдёӘsegment fileпјҢжүҖд»Ҙй»ҳи®Өжғ…еҶөдёӢжҳҜ1з§’й’ҹдёҖдёӘsegment fileпјҢsegment fileдјҡи¶ҠжқҘи¶ҠеӨҡпјҢжӯӨж—¶дјҡе®ҡжңҹжү§иЎҢmergeгҖӮ
		
		5.13пјҡжҜҸж¬Ўmergeзҡ„ж—¶еҖҷпјҢдјҡе°ҶеӨҡдёӘsegment fileеҗҲе№¶жҲҗдёҖдёӘпјҢеҗҢж—¶иҝҷйҮҢдјҡе°Ҷж ҮиҜҶдёәdeletedзҡ„docз»ҷзү©зҗҶеҲ йҷӨжҺүпјҢз„¶еҗҺе°Ҷж–°зҡ„segment fileеҶҷе…ҘзЈҒзӣҳпјҢиҝҷйҮҢдјҡеҶҷдёҖдёӘcommit pointпјҢж ҮиҜҶжүҖжңүж–°зҡ„segment fileпјҢз„¶еҗҺжү“ејҖsegment fileдҫӣжҗңзҙўдҪҝз”ЁпјҢеҗҢж—¶еҲ йҷӨж—§зҡ„segment fileгҖӮ
		
		esйҮҢзҡ„еҶҷжөҒзЁӢпјҢжңү4дёӘеә•еұӮзҡ„ж ёеҝғжҰӮеҝөпјҢrefreshгҖҒflushгҖҒtranslogгҖҒmerge
		
		еҪ“segment fileеӨҡеҲ°дёҖе®ҡзЁӢеәҰзҡ„ж—¶еҖҷпјҢesе°ұдјҡиҮӘеҠЁи§ҰеҸ‘mergeж“ҚдҪңпјҢе°ҶеӨҡдёӘsegment fileз»ҷmergeжҲҗдёҖдёӘsegment fileгҖӮ
