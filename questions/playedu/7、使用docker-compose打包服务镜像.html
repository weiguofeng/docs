    # 4、清理Docker缓存-磁盘空间不够

    清理Docker缓存有时可以解决构建问题：

    docker system prune -a
    1

    这将删除所有未使用的数据，包括所有停止的容器、所有未使用的网络、所有未被一个以上容器所使用的镜像和所有未使用的构建缓存。
