# 2、音标核心要点



​	我们使用的是DJ音标



## 1、元音

​			区分各个元音的3要素是

​			1、舌位的前后

​		2、舌位的高低

​		3、嘴唇的 “圆唇度”

#### 3、双元音

​			双元音是由多个单元音组合成的

​			其实咱们普通话的双韵母就是双元音

​	我们可以把双元音理解为 就是2个单元音 从前一个快速滑动到后一个发出的音

![1715851496956](../../.vuepress/public/images/1715851496956.png)
