### 2.JSX的基本使用

```
	JSX是JavaScript XML的简写，表示在JavaScript中写XML（HTML）格式的代码。
	优势：声明式语法更加直观，与HTML结构相同，降低了学习成本，提升开发效率。
	JSX是React的核心内容。
	
	
```
