​		使用supper()关键字调用父类的构造函数，可以看到我们需要传递一个布尔值true或者false，这并不重要，重要的是，这份声明应该是这个构造函数中的第一条语句。

![image-20220409222427764](../../../../../.vuepress/public/images/image-20220409222427764.png)
