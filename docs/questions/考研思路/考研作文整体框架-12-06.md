# 考研作文整体框架



#### 	1、小作文

​		小作文大致分俩类，1、公务书信 2、私人书信



​	公务书信就是给不认识的人写信，私人书信就是比较亲密的人写信，比如朋友之类的。如果是公务书信的话，第1句话就自我介绍一下，如果题目给出了你的身份，题目怎么说的，你就怎么介绍自己
